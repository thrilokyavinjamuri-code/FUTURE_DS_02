# Customer Churn Analysis Dashboard

## Objective
Analyze customer churn patterns using Power BI.

## Tools Used
- Power BI
- CSV Dataset

## KPIs
- Total Customers
- Churn Customers
- Churn Rate
- Active Customers

## Dashboard Features
- Churn by Gender
- Churn by Contract Type
- Churn by Internet Service
- Monthly Charges Analysis
- Interactive Filters

## Key Insights
- Month-to-month customers showed highest churn.
- Fiber optic users experienced higher churn.
- Higher monthly charges increased churn probability.
- Long-term customers were more likely to stay.
