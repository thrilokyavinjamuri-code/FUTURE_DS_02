# Customer Churn Analysis Dashboard

## Objective

Analyze customer churn patterns using Power BI and identify factors affecting customer retention.

## Tools Used

* Power BI
* CSV Dataset

## KPIs

* Total Customers
* Churn Customers
* Churn Rate
* Active Customers

## Dashboard Features

* Churn by Gender
* Churn by Contract Type
* Churn by Internet Service
* Monthly Charges Analysis
* Interactive Filters & Slicers
* Key Insights Section

## Key Insights

* Month-to-month customers showed the highest churn.
* Fiber optic users experienced higher churn rates.
* Customers with higher monthly charges were more likely to churn.
* Long-term contract customers were more likely to stay with the company.
